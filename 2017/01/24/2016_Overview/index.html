2016总结

发表于 2017-01-24   |  

2016年 总体来说，生活上的事情已经都告一段落了，工作上却是一个新的开始，移动互联网方向的热度大不如前了，但这条路还要继续走下。后续需要对移动端的各项技术都要了解，不论iOS、Android，还是HTML5 等等。

上半年生活的变化比较多，罗列了下：

1.  房子的装修已完成，5月也入住了，算是在大杭州有个窝的人了；
2.  5月在老家把婚礼也举办了，人生第一次喝醉、喝断片，但很开心；
3.  结束了在树熊的工作，很感谢熊孩子们两年来的共事；
4.  6月进入新公司，还是物联网(智能硬件方向)，看样子还要在这个圈子里继续厮混；

下半年的生活呢给了我警醒，家里老人的突然生病和老婆怀上宝宝，让我对生活的压力有了重新的认识 — 担子更沉重了。
生活也不再是我和老婆两个人亲亲我我了，即将出世的小宝宝和来照顾宝宝的老妈或者丈母娘，将要和我们一起生活。

工作上，新的公司，新的同事，新的环境。
同事们都很好，码农嘛，相互之间也没有什么界限，一起吃饭，互相吹吹牛，偶尔拆拆台，不深入，但也很友善。
公司在业务上暂时处于领先的地位，但是追赶者众多，而且受制于大厂的地方较多，整体还是一个业务驱动的公司；
在人文上就要吐槽下了，没有了之前熊孩纸的福利，处处感觉到抠的状态，幸福指数直线下降；
在工作上，不在单纯的搞iOS了，Android，JNI，Lua, Linux C各个方面都需要涉及，
具体就是做客户端SDK开发，由于SDK还对外发布，给客户使用，在此过程中还接触了形形色色的客户，吐槽…

2017 期待 责任 潜心 前行
