2017年总结

发表于 2018-02-10   |  

不知不觉又走到了2017年度最后一个工作日了，翻开好久没更新的博客，惭愧已经一年多没有更新内容了…
补上一篇年度总结算是对这一年度的补救和对下一年度的期待吧。

生活

2017.5.12 宝宝终于出生了，是个大胖小子，萌萌哒。每天回家看到这小子，就能感觉到发自内心的愉悦。
生活的重心都已经倾向于照顾这小子了，每次出门都是匆匆而去，匆匆而回，购物总是去婴幼儿店里看看是否
需要什么要买。

工作

公司

好的方面：公司今年发展很快，和各个大公司合作更加紧密，年末又宣布了D轮融资，明年也即将搬入新的办公楼 😄

差的方面：人性化上还是那么多的槽点，管理依旧原始和混乱，还有那可怜的福利…

团队

今年开始负责 SDK 团队，虽然团队很小，但内心里认为这是个核心团队，至少在核心业务的推进上起到关键的作用。
团队上开始推行 Teambition 进行管理，每隔两周做一次技术分享和交流，偶尔大家一起开个黑。

业务

今年智能音箱大火，有幸能从一开始就参与其中，从年初亚马逊的 Echo ，双十一的天猫精灵，再到近段时间的度秘，

我们的 SDK 都能第一时间完善的接入其中，并且能发展成客户对接的标准，内心还是点点自豪。
今年的出差也很多，在淘宝城呆了小半个月，再到去珠海、深圳、香港来来回回的飞，都是为了能够更好的让 SDK 接入这些大厂。

个人

iOS 方面，今年重点学习了 Swift 和 RxSwift，虽然在工作中暂时还无法切换过去，但已经开始慢慢使用这两个重写原有的APP了。

Android 方面，好吧，我还是个菜鸟。

Web 端，因为业务上的需要，许多Native都切换到了H5，SDK也开始支持H5的直接调用，舍弃了原有庞大臃肿的Cordova，开始构建自身的JSBridge。

展望

2018 钻研 奋斗 更进一步
